Q1:有哪些你熟悉的监督学习和无监督学习算法？

  • 监督学习：
    1. 线性回归
    2. 逻辑回归
    3. SVM（SVC && SVR）
    4. 决策树
    5. 朴素贝叶斯
    6. 神经网络
  • 无监督学习：
    1. K-means
    2. PCA

Q2:生成模型和判别模型的区别？

  • 判别模型：
    是直接根据现有的数据，通过数据集给定的X，学习到相应的决策函数，来预测Y；
  • 生成模型：
    是先学习到数据集上的联合概率分布情况，然后，再进一步从联合概率的来预测Y。

Q3:线性分类器与非线性分类器的区别以及优劣

  • 线性分类器：通过线性函数来拟合数据，即默认数据是线性可分的，优势在于：Hypothesis简单，方便求解；劣势在于：无法对线性不可分的数据集进行很好的拟合；
  • 非线性分类器：通过非线性函数来拟合数据，将数据映射到高维空间中，实现在高维空间中的可分。优势在于：可处理线性不可分的数据集，且拟合效果好；劣势在于：与线性分类器相比，求解更加复杂。

Q4:什么是交叉验证，交叉验证如何使用在时间序列上？

交叉验证是将数据集切成N份，进行N次试验：选取其中的N-1份作为训练集，剩下的1份作为测试集。

Q5:分类和回归里有哪些你熟悉的评估准则，分别是什么样的含义？

  • 回归评价指标：

    1. MSE：均方误差，表示的含义为：是预测值-真值间偏差平方和与样本数量的比值，MSE越小，表示模型的预测效果越好。
    2. RMSE：均方根误差，为MSE的开根号。与MSE的区别就在于开根号后，与原数据的量纲相同。
    3. MAE：平均绝对误差，表示的含义为：是预测值-真值间偏差绝对值和与样本数量的比值。
    4. $R^{2}$:R-squared。是MSE/var(y),表示模型对数据的拟合效果的好坏，取值在0-1，越大表示模型效果越好。

  • 分类评价指标：

    1. 精度&误差率：精度是分类错误的数量占总数的比例；误差率 =（1-精度）
    2. 准确率P：TP/(TP+FP)
    3. 召回率R：TP/(TP+FN)
    4. $F_{\beta}$:$\frac{(1+\beta^2)\times P \times R}{(\beta^{2} \times P)+R}$;$\beta$小于1时，准确率影响更大；$\beta$大于1时，召回率更有影响。
    5. AUC：预测为正例的概率值比预测为负例的概率值还要大的可能性。

Q8:遇到缺失值，我们有哪些处理方法？

根据缺失值缺失的数量来看：

  • 如果某一维度上，缺失值数量很大，那么，则可直接将该维度直接删除；

  • 如果缺失值数量较小，一般用填充的方法，如中位数、众数等。

Q9:你了解哪些特征工程的处理？

从数据中抽取出对结果预测有用的信息。

  • 数据预处理：
    1. 数值型：
      • 归一化（min-max）/标准化（z-score）
      • 离散化（分桶）/二值化（小于、大于阈值）
      • log变换、Box-Cox变化（将数据转为趋近于正态分布）
    2. 类别型：
      • 序号编码（label encoding）
      • one-hot编码
      • 特征哈希处理

Q10:逻辑回归中为何选择sigmoid函数，这个函数有什么特点？

逻辑回归，假设了预测Y|X是服从贝努利分布的，将其写成指数族分布的形式，在广义线性模型的假设下，可推导出sigmoid函数。

sigmoid函数的特点：

  • 严格单调递增，且可导；
  • 关于（0,0.5）对称；
  • 值域范围限制在(0,1)之间;
  • 导数为：$f’(x)=f(x)(1-f(x))$；计算非常方便。

Q12:讲解一下决策树的建模方法，对于连续值属性它是如何处理的？

决策树的建模：

（1）选择分裂点：在不同算法中，评价指标不同：信息增益,信息增益比,Gini指数；

（2）树的生长：按照（信息增益最大/信息增益比最大/Gini指数最小）的原则循环迭代不断建立树的子树，将子树中最多出现的类别作为当前分值的类别标记；

（3）停止条件：所有样本都属于同一类别。

连续值的处理：将该属性上的取值从小至大排列，用每两个数之间的均值，即n-1个离散点将连续值切分为n段。

Q13:随机森林是一个什么模型？

是一个监督学习下的集成学习模型。

通过boostrap sampling进行采样，从样本总采样出T个样本子集，以增加样本扰动；在树的分裂过程中，从待分裂属性中，随机选取部分属性作为属性子集，在属性子集中进行最佳分裂点选取，以增加属性扰动。

最后，分类通过投票、回归通过简单平均得出最后的集成结果。

Q14:你了解哪些boosting的模型，Xgboost的原理是什么？

Boosting模型：是串行集成的集成学习模型，有：AdaBoost和GBDT。

  • AdaBoost：通过计算上一颗树的误差来进行重新采样，改变样本的分布（倾向于分错的样本）再进行训练。

  • GBDT：将上一颗树的结果与真值的偏差作为下一颗树的输入，不断迭代减小模型预测值的偏差。

  • Xgboost原理：在GBDT的基础上，对于目标函数增加了惩罚项（叶子节点数+叶节点权重的L2正则），并对目标函数在$\hat{y}^{(t-1)}$处进行二次泰勒展开，将其从以样本进行的求和转换为以叶节点进行的求和，这样就将原目标函数转化为了二次方程求极值的问题。

Q15:Xgboost和GBDT的主要区别是什么？

  1. Xgboost优化方法使用了二阶导数，GBDT优化方法使用一阶倒数；
  2. Xgboost相对于GBDT，增加了惩罚项：（叶子节点的个数，叶子节点的权值L2正则）
  3. XgBoost使用了Shrinkage策略，降低每次学习的效果，防止过拟合
  4. XgBoost对特征的缺失值有特别的处理方法

Q15.1:Xgboost如何处理多分类问题？

应用OvR方案，即针对于N个分类，分别训练出N个分类器，各分类器的训练内容为：属于当前分类或不属于当前分类。

Q15.2:Xgboost的缺失值处理逻辑是什么样的？

对于缺失值，其当做稀疏矩阵，在节点分裂的时候，会被分到左子树和右子树，根据损失选择较优的那一种情况。对于在训练没有缺失值，预测时出现缺失值时，默认被分到右子树中。

Q15.3:Xgboost如何计算特征重要度？

Xgboost特征评分：可以看成是被用来分离决策树的次数，次数越多，那么重要度越大。

Q16:怎么判断模型过拟合，缓解模型过拟合的方法有哪些？

过拟合的表征：在训练集的效果非常好，但是在测试集中效果一般。

缓解过拟合的方法：

  • 增加正则化项；
  • 调整参数，以降低模型的复杂程度；
  • 增加数据；
  • 对于树模型可采用shrikage；对深度学习可采用Dropout

Q17:L1和L2正则化的区别是什么，如何理解？

L1正则化相比于L2正则化，可以起到对特征筛选的效果。

因为L1正则是参数的绝对值之和，在训练时，会很容易将对模型影响较小的特征的参数变为0，而L2正则不会。

从贝叶斯的角度看：L1正则对参数加入了拉普拉斯分布的先验证，L2正则对参数加入了高斯分布的先验，那么，显然，拉普拉斯分布是在参数为0时的概率密度极大，高斯分布是在0附近的概率密度都很大，显然，二者相比，L1会更容易产生参数为0的。

Q18:CNN的不同层次处理和功能介绍？

  • 卷积层：对数据提取局部特征；
  • 池化层：对卷积层提取到的特征进行下采样；
  • 全连接层：在整个网络中起到了一个分类器的作用；
  • SoftMax：上一层输出的特征向量来计算属于各分类的概率。

Q19:CNN里的1*1卷积有什么作用？

它的作用是：

  • 在不大幅改变网络性能的前提下，减少运算量（瓶颈层嘛）。
  • 升维/降维（通道的维度）：因为卷积核的大小是1*1的，所以，并不会改变原矩阵的长和宽（padding会改变），用k个1*1的卷积核会改变通道的数量；
  • 跨通道信息交换：比如原channels数量为C1，用C2个1*1*C1的卷积核，我们可以认为是在通道上从C1到C2的变化，实现通道间的信息交换；
  • 增加非线性。

残差网络的残差模块是什么样的？

Inception模块其实就是各种尺寸的卷积核的mix。

构成如下：（并列结构）

（1）n个1*1的卷积核（瓶颈层）+不同尺寸的卷积核：1*1卷积核降低通道数量，不同尺寸的卷积核提取不同大小的特征；

（2）maxpooling + k个1*1的卷积核（瓶颈层）：pooling并不改变通道数量，所以用1*1的卷积核（瓶颈层）来降低通道数量；

最后将上述得到的各通道叠加起来，就Over了。

LSTM为什么能缓解梯度弥散问题？

LSTM在模型中设置了记忆细胞，通过更新门和遗忘门来选择对历史时刻的信息存储。其在时间序列上，记忆细胞的值也进行传递，其传递的值取决于上一时刻的记忆值和当前的候选值，梯度下降计算时是对上述二者的门进行的，因此，在整个时间序列上，很好地保存下了长期依赖关系，缓解梯度弥散。
